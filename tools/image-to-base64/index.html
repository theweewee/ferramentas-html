图片 Base64 互转

图片与 Base64 编码字符串之间的双向转换

关于这个工具

图片 Base64 互转工具可以实现图片文件与 Base64 编码字符串之间的双向转换。Base64 编码的图片可以直接嵌入 HTML、CSS 或 JavaScript 中，无需额外的 HTTP 请求。同时也可以将 Base64 字符串还原为图片文件。

功能特点

  • 支持图片转 Base64 编码
  • 支持 Base64 字符串转回图片
  • 支持拖拽上传图片
  • 支持 JPG、PNG、GIF、WebP 等常见格式
  • 实时预览图片
  • 显示图片尺寸和文件大小信息
  • 输出 Data URI 或纯 Base64 字符串
  • 支持 Data URI 和纯 Base64 两种输入格式
  • 一键复制或下载 Base64 数据
  • 一键下载转换后的图片
  • 完全在浏览器本地处理，保护隐私

使用场景

  • 在 HTML 中嵌入小图标
  • CSS 背景图片内联
  • 减少 HTTP 请求数量
  • Email 中的图片嵌入
  • 单文件 HTML 应用中的图片
